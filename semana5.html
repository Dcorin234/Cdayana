CLASE VIRTUAL

En clase conocimos un mix de plataformas y herramientas online para diseñar páginas web sin complicaciones,como ejemplo Wix,Canva.En resumen tenemos que CANVA Y GOOGLE SITES ofrecen enfoques diferentes, Canva ofrece plantillas de diseño web que puedes utilizar como punto de partida para crear tu sitio web. Estas plantillas suelen estar diseñadas para páginas de aterrizaje o sitios web sencillos.Por otro lado GOOGLE SITES,Google Sites se enfoca en la creación de sitios web completos y funcionales. Puedes crear sitios web con múltiples páginas, menús de navegación y contenido variado.
